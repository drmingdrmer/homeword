如：
function fruit(name, size, color){
....
}

var apple = fruit("apple", 20, "red");
这时，
apple.name是"apple", apple.size是20，apple.color是"red"
你看到那3个相当于了吧？
那实际上是函数调用返回后生成的1个中间步骤。
函数调用的部分会被替换成它的返回值，然后再继续执行。
再详细1点,这样：
document.write(fb(1));
遇到fb(1)，首先是函数调用，传递1个参数1
这时这句的执行会先暂停。
跳转到fb函数内部。
在函数内部相当于执行了这样1段代码：
var i=1;
return i*2;
这段执行结束之后，函数fb的调用结束。
返回到document.write(...这里。然后把返回回来的值放到这里，代替刚才的函数调用fb(1)这部分。
变成document.write(2);
